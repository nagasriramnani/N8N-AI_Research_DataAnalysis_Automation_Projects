## üöÄ Getting Started

### Prerequisites

- [n8n](https://n8n.io/) installed (self-hosted or cloud)
- API keys for required services (OpenAI, Google, etc.)
- Vector database access (Qdrant, Pinecone) for RAG workflows

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/nagasriramnani/N8N-AI_Research_DataAnalysis_Automation_Projects.git
   ```

2. **Import workflows into n8n**
   - Open your n8n instance
   - Go to **Workflows** ‚Üí **Import from File**
   - Select the `.json` file you want to import

3. **Configure credentials**
   - Set up required API credentials in n8n
   - Update any hardcoded values to match your environment
